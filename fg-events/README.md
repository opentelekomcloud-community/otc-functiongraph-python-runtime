# fg-events

Event definitions for FunctionGraph triggers.

## Install all events from fg_events

To install dependencies run:

```
pip install git+https://github.com/opentelekomcloud-community/otc-functiongraph-python-runtime.git@main#subdirectory=fg-events
```


Or add following statement to your  ```requirements.txt```

```
fg-events @ git+https://github.com/opentelekomcloud-community/otc-functiongraph-python-runtime.git@main#subdirectory=fg-events
```

and run:

```
pip install -r requirements.txt
```
