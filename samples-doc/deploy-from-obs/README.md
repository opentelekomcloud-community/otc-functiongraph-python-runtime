# deploy-from-obs

This sample demonstrates how to deploy FunctionGraph with **code from OBS** using terraform.

This approach is used, if your unpacked FunctionGraph deployment package is **greater than 40MB**.

For details see [Developer Documentation](https://opentelekomcloud-community.github.io/otc-functiongraph-python-runtime/devguide/deployment/terraform/deploy-from-obs/_index).
